# Book App

This directory contains the Docusaurus frontend for the Physical AI & Humanoid Robotics textbook.

## Development

1.  Navigate to this directory:
    ```bash
    cd book-app
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Start the development server:
    ```bash
    npm run start
    ```

The site will be available at `http://localhost:3000`.

## Adding Content

-   All book content is written in Markdown (`.md` or `.mdx`) files inside the `/docs` directory.
-   The sidebar navigation is configured in `sidebars.ts`.
-   Follow the established structure of Parts and Chapters when adding new content.
